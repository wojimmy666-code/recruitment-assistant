招聘助手便携版
================

一、启动服务器
1. 解压整个目录，不要只复制其中部分文件。
2. 双击 start-server.bat。
3. 等待浏览器自动打开 http://localhost:3000。
4. 如需停止服务器，双击 stop-server.bat。

二、安装 Chrome 扩展
1. 打开 chrome://extensions。
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录下的 chrome-extension 文件夹。
5. 固定扩展图标，方便在 BOSS 页面使用。

三、开始使用
1. 在招聘助手页面保存职位和筛选条件。
2. 使用正常 Chrome 登录 BOSS 招聘者后台。
3. 打开 https://www.zhipin.com/web/chat/recommend。
4. 点击招聘助手扩展图标并开始批量打招呼。

四、数据迁移
所有本地数据保存在 data\app.sqlite。
迁移前先在旧电脑运行 stop-server.bat，然后复制 app.sqlite 到新电脑的 data 目录。
不要复制 app.sqlite-wal 或 app.sqlite-shm 临时文件。

五、故障排查
- 端口被占用：关闭占用 3000 端口的程序后重试。
- 页面无法打开：查看 logs\server.error.log。
- 扩展无法连接：确认服务器已启动，并重新加载 Chrome 扩展。
- Windows 安全提示：确认文件来源后允许本地 Node.js 程序运行。
